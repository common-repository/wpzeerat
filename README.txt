== Description ==

[Zeerat](https://www.zeerat.com/) helps people to understand visitorsâ€™ behavior on a website, reveals bugs and errors of a website conversion strategy such as uncompleted goals (form and cart abandonment) or navigation and design problems.

== Installation ==

= Automated Installation =

1. Go to the "Plugins" menu, and choose "Add New".
2. Search for "wpzeerat", and then click "Install Now".
2. Click "Activate".

= FTP =

1. Download and extract the .zip file.
2. Upload the unzipped folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu.

= Composer =

`composer require s3rgiosan/wpzeerat`

= Configuration =

After installation, configure the plugin by visiting Settings -> Zeerat and entering your snippet code.
